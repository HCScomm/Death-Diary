# 行为包特色
###### *游戏时间*将与*现实世界相同*。 
###### 你*不能睡觉*。 
###### 这个世界*不存在有好生物*。（除了NPC） 
###### 你可以和游戏中随机生成的*NPC*进行物品交易。 *（即将到来）* 
###### *随着时间变化*僵尸的装备会*越来越强*。 
###### 每一只僵尸的*强度是随机的*。 
###### 世界的天气会*随着季节更替而出现几率性*。 
###### 你会有一个新的值——*体温*。 
###### 会出现*季节更替*。

###### 解药*研发完毕*世界就会*恢复原样*。 *（即将到来）* 
###### 你*不能死亡*,一旦死亡就会*游戏结束*。 *（即将到来）*
